cairo-picker (カイロピッカ)


GitHub
リリース日 : 2020/07/08
使用言語 : C++ (g++)
開発環境 : Ubuntu (19.10)

ウインドウ上にベクタグラフィックスを描画するための最小限の機能を実装したライブラリ。
描画内容のプログラミングにはCairoの関数を用いる。
プログラムのビルド時、Gtkmm(gtk3のC++ラッパー)が必要である。

最低限の機能しか用意していないため、描画以外の機能の自由度には欠けるが、使用時に覚えることが少ない上動作が非常に軽量である。
ループ描画を行えるため、動きのあるグラフィックスを描画することができる。
またキー入力やマウス入力にも対応しているため、GUIアプリケーションやゲームなどを作ることも可能。

現在は、主に研究内容の試験実装の検証に用いている。

使用例

#include "cairo_picker.hpp"

Capr::C_picker picker;
int x = 0;

// 描画内容を設定する関数 void func(Capr::Cairo_cont cr){
 cr->set_source_rgba(1.0, 1.0, 1.0, 1.0);
 cr->paint();
 cr->set_source_rgba(1.0, 0.0, 0.0, 0.5);
 cr->rectangle(x++,100,150,150);
 cr->fill();
}

int main(){
 // ウインドウのサイズを設定
 picker.set_default_size(400,400);
 // ウインドウの再描画の頻度を更新(下の場合、10/60 ≒ 0.167秒に一回ウインドウが再描画される)
 picker.set_loop(10);
 // アプリケーションを実行 (関数funcを繰り返し実行)
 picker.run(func);
}
